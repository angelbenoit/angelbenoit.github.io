Bart App

Bart app is a front end web application using the BART api to get live data. Users can view incoming and departing trains from any train. They'll also be able to view where the trains are headed, how many cars, and in how many minutes it'll arrive to the specified station. Users will also be able to view routes from point A to point B. Additionally, they'll be informed whether user will need to transfer trains. Lastly, users can view more information about any station in the BART system.
